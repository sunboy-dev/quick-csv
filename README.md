# Quick CSV

Quick CSV is a fast, local, and user-friendly web application for viewing CSV data. It works entirely in your browser without uploading your data to any external server.

---

## 📖 วิธีใช้งาน (How to use)

แอปพลิเคชันถูกออกแบบมาให้ใช้งานง่าย รวดเร็ว และไม่ต้องติดตั้งโปรแกรมเพิ่มเติม:

1. **นำเข้าข้อมูล (Import Data):** 
   - คัดลอกข้อความ CSV แล้วนำมา **วาง (Paste)** ลงในกล่องข้อความฝั่งซ้ายมือ
   - ลากไฟล์ `.csv` มา **วาง (Drag & drop)** ลงในพื้นที่อัปโหลดฝั่งขวามือ
   - หรือ **คลิก (Click)** ที่พื้นที่อัปโหลดเพื่อเลือกไฟล์ `.csv` จากในเครื่อง

2. **ดูข้อมูลและตรวจสอบ (View Data):**
   - เมื่อข้อมูลถูกนำเข้าเสร็จสิ้น ระบบจะประมวลผลและแสดงตารางให้โดยอัตโนมัติที่ด้านล่าง

3. **ค้นหาข้อมูล (Search):**
   - พิมพ์คำหรือข้อความที่ต้องการค้นหาในช่อง **Search...** ทางขวาบนของตาราง
   - ข้อมูลในตารางจะถูกกรองให้เหลือเฉพาะแถวที่ตรงกับคำค้นหาทันที

4. **เรียงลำดับข้อมูล (Sort):**
   - คลิกที่ชื่อ **หัวคอลัมน์ (Column Header)** เพื่อเรียงลำดับข้อมูล
   - **คลิกครั้งที่ 1:** เรียงลำดับจากน้อยไปมาก (Ascending)
   - **คลิกครั้งที่ 2:** เรียงลำดับจากมากไปน้อย (Descending)
   - **คลิกครั้งที่ 3:** ยกเลิกการเรียงลำดับ (กลับสู่ค่าเริ่มต้น)

---

## 🛠 สำหรับนักพัฒนา (Development)

หากต้องการรันโปรเจกต์นี้ในเครื่องตัวเอง (Local) สามารถทำตามคำสั่งด้านล่างได้เลยครับ:

```bash
# ติดตั้ง Packages ที่จำเป็น
npm install

# รัน Development Server
npm run dev
```
